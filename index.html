ETF Holdings Aggregator

Upload one or more CSV files downloaded from ETF websites (BOUT, FFTY, QMOM, etc.). This tool will extract all tickers and aggregate them into a single comma-separated list (no quotes).

You can select multiple files at once (e.g., BOUT.csv, FFTY.csv, QMOM.csv). The tool auto-detects the ticker column (looks for headers like Ticker/Symbol, otherwise guesses by pattern).

Comma-separated tickers

Unique tickers from all uploaded files (sorted alphabetically, no quotes).
