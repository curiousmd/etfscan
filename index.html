ETF Holdings Aggregator

Fetch holdings for BOUT, FFTY and QMOM from Financial Modeling Prep, then aggregate all tickers into a single comma-separated list.

Source: financialmodelingprep.com/api/v3/etf-holder

The API key is hard-coded in this file. Use this only for personal/local use and avoid sharing the file with your real key.

Per-ETF holdings

Each card shows the tickers returned for that ETF by /api/v3/etf-holder/<symbol>.

Aggregated tickers

Unique tickers across BOUT + FFTY + QMOM (sorted alphabetically).
